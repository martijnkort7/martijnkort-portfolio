# Portfolio — Martijn Kort

## Project
Persoonlijke portfoliosite gebouwd met Next.js 15, TypeScript en Tailwind CSS v4.
Geïnspireerd op brittanychiang.com: donker thema, sticky navigatie, smooth scroll.

## Tech stack
- Next.js 15 (App Router, geen src/ dir)
- TypeScript (strict mode)
- Tailwind CSS v4
- Framer Motion voor animaties
- Vercel voor deployment

## Skills
Dit project gebruikt de **frontend-design** skill (Anthropic). Volg bij elk component
de richtlijnen uit de skill: kies een bewuste esthetische richting, vermijd generieke
fonts (geen Arial, geen standaard Inter), en zorg voor doordachte animaties en compositie.

## Conventies
- Alle componenten in app/components/
- Responsive: mobile-first
- Taal van de content: Nederlands (behalve sectienamen: About, Experience, etc.)
- Gebruik semantic HTML en ARIA-labels voor accessibility
- Commits in het Engels, imperative mood ("Add hero section", niet "Added hero section")
- Geen console.logs in productie

## Design
- Esthetische richting: refined dark, geïnspireerd op brittanychiang.com
- Kleurenpalet: donkere achtergrond (#0a192f of vergelijkbaar), lichte tekst (#ccd6f6), accent (teal/cyan #64ffda)
- Typografie: kies een karakteristiek display font voor headings (geen Inter/Arial/Roboto), een clean body font
- Sticky navigatie links in de sidebar (desktop), hamburger menu (mobiel)
- Smooth scroll naar secties
- Animaties: framer-motion voor scroll-triggered reveals en staggered entry. Focus op een paar sterke momenten, niet overal kleine animaties
- Achtergrond: voeg diepte toe met subtiele gradient, noise texture, of grain overlay — geen platte effen kleur
- Hover-states: elke interactieve element moet een doordacht hover-effect hebben
